# AI Care Simulations

Prototype for AI-based behavioral scenario training in neurodivergence care.

---

## What It Does

The learner receives:
- a behavioral scenario
- writes a response
- receives competency-style feedback

---

## Current Prototype

- 5 behavioral scenarios
- response input
- competency scoring
- empathy detection
- reinforcement detection
- safety evaluation
- next-step analysis

---

## Why This Matters

Care professionals need:
- practice
- simulations
- repetition
- feedback

not just passive video content.

This prototype explores the idea of:
# a flight simulator for human care.

---

## Long-Term Vision

Future versions may include:
- AI evaluators
- adaptive learning paths
- competency graphs
- ABAT/RBT readiness scoring
- supervision infrastructure
- workforce progression systems
